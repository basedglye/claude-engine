# Living World (flagship — Phase 4)

A persistent 3D living world built on ClaudeEngine, intended to go past
World of Claudecraft:

- MMO-style foundation: zones, quests, parties, economy — on engine-owned
  netcode and persistence instead of generated one-offs.
- Claude-driven NPCs and world events (Agent SDK) behind a governance layer:
  schema-constrained outputs, approval gates for world-affecting changes.
- An autonomous improvement loop where player feedback becomes
  harness-verified pull requests.

Empty for now on purpose: the engine (Phases 1–3) comes first. See
[docs/ROADMAP.md](../../docs/ROADMAP.md).
